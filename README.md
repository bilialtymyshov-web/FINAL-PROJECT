# FastBoard — Сайт объявлений с фото и ценами

Финальный учебный проект на Django для демонстрации навыков веб-разработки.

---

## 📝 Описание проекта
**FastBoard** — это полноценная онлайн-доска объявлений. Пользователи могут публиковать свои товары, просматривать витрину актуальных предложений, фильтровать товары по категориям и диапазону цен, а также управлять своими публикациями.

---

## 🌟 Реализованный функционал

### Базовый функционал (Must-have):
* 🔐 **Авторизация и безопасность:** Интеграция со стандартной системой пользователей Django. Защита маршрутов через `LoginRequiredMixin` и `UserPassesTestMixin`.
* 📦 **Модели данных:** Категории (`Category`), Объявления (`Ad`) с привязкой к авторам и категориям.
* 🛍️ **Витрина объявлений:** Главная страница со списком только активных товаров.
* 🔍 **Поиск и фильтрация:** Поиск по названию, фильтрация по категориям и диапазону цен (`min_price`, `max_price`).
* 📄 **Детальная страница:** Подробная информация об объявлении, авторе, дате публикации и категории.
* ⚙️ **Полноценный CRUD:** Возможность создавать, редактировать и удалять собственные объявления.
* 👤 **«Мои объявления»:** Отдельный личный кабинет для просмотра своих публикаций.

### Челлендж-функции (Бонусные фишки):
* 👁️ **Счётчик просмотров:** Автоматический инкремент просмотров при посещении детальной страницы (с использованием `F()`-выражений для исключения race condition).
* 🏷️ **Статус объявления:** Переключение состояния (`Активно` / `Продано`).
* 🖼️ **Галерея фото:** Поддержка главной обложки и загрузки дополнительных фотографий через связанную модель `AdImage`.

---

## 🛠️ Технологии
* **Backend:** Python 3.10+, Django 4.x
* **Database:** SQLite
* **Frontend:** Bootstrap 5, HTML5, CSS3
* **Libraries:** Pillow (для работы с медиафайлами)
